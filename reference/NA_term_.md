# Missing Term

A missing term element.

## Usage

``` r
NA_term_
```

## Format

An object of class `term` (inherits from `vctrs_vctr`) of length 1.

## See also

[`term-vector()`](https://poissonconsulting.github.io/term/reference/term.md)

## Examples

``` r
is_term(NA_term_)
#> [1] TRUE
is.na(NA_term_)
#> [1] TRUE
```
