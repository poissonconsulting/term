# Missing Term

A missing term element of `term_rcrd` type.

## Usage

``` r
NA_term_rcrd_
```

## Format

An object of class `term_rcrd` (inherits from `vctrs_rcrd`,
`vctrs_vctr`) of length 1.

## See also

[`term-vector()`](https://poissonconsulting.github.io/term/reference/term.md)

## Examples

``` r
is_term_rcrd(NA_term_)
#> [1] FALSE
is.na(NA_term_)
#> [1] TRUE
```
